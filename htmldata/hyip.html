// Маркетинг и финансовый план хайпа

Для обеспечения гарантированного спроса на монету, создается проект, который принимает именно эту монету.

Почему?

Потому то основная задача этой монеты - быть алгоритмом, обеспечивающим исполнение правил, установленных для проекта, принимающего эту монету, плюс неизменяемой базой данных событий, происходящем в этом проекте, плюс регистрацию и идентификацию участника.
