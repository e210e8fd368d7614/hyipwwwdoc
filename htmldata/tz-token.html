/* Техническое задание на создание сервиса продажи токена */

Требуется создать адекватный и удобный механизм продажи токена проекта.
Этот механизм должен соответствовать нескольким условиям:

1. Токен должен генерироваться на блокчейне монеты, торгуемой на большом количестве бирж

2. Доступ к покупке токена должен осуществляться со страниц этой документации

3. Цена на токен должна выставляться по алгоритму, позволяющему осуществлять её плавный рост, диктуемый арифметической прогрессией, начиная от 0,1% конечной рассчетной стоимости

4. Обеспечить механизм продажи токенов других участников, предусматривающий очередность, определяемую по дате приобретения токена участником
