Наброски технического задания для кодеров.

Требования к программистам.
Знание любых пакетов для кодинга под смартфоны и персональные компьютеры.
Навскидку: создаваемое приложение будет работать на гаджетах под управлением операционных систем Андроид, Win10, Win7, iOS, Ubuntu, Debian, Centos, macOS.

Перечень систем и ПО, требуемых для функционирования проекта

  1. Криптовалюта с генерацией нарастающего количества монет в блоке, pow майнингом, на алгоритме, обеспечивающем полную защиту от майнеров, ориентированном на процессоры бытовых гаджетов.

  2. Кошелек, оснащенный функциями майнера, бухгалтерии, пейджера.

  3. Личный кабинет, обеспечивающий анонимность, возможность управления всеми требуемыми функциями хайпа.

  4. Бухгалтерия проекта для сотрудников проекта
